## ESP32 Mini Flight Computer 25-26

#### Prerequisites
- ESP-IDF v5.5.x (using the ESP-IDF extension on VSCode is recommended for ease of use)
- VSCode or any IDE with C/C++ Support
- ESP32S3 Microcontroller board (to flash firmware)

#### ESP-IDF Official Installation Guide: 
https://docs.espressif.com/projects/esp-idf/en/latest/esp32/get-started/index.html

#### STM32 I2C Sensors:
- LSM6DSV80X: 6-axis IMU (+/- 80g Accelerometer + Gyroscope)
- LPS22H: Barometer / Pressure Sensor
- IIS2MDC: Magnetometer
